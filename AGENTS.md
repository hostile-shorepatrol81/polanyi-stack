## Project

Tacit Skills is an open-source framework for converting tacit knowledge, expert judgment, and Michael Polanyi style tacit knowing into reusable AI agents, Codex skills, and training workflows.

## Repository Purpose

Use this repository when the user wants to:

- understand tacit knowledge
- convert tacit knowledge into an AI agent
- convert expert judgment into a Codex skill
- design onboarding or apprenticeship for judgment-heavy work
- build reusable workflows from hard-to-explain expertise

## Short Answer

The repository's central claim is that the most valuable operating logic in a team often lives in tacit knowledge rather than explicit documentation. The practical job is to extract that hidden layer and package it into reusable agents, skills, or training systems.
